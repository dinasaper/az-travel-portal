# 🌐 AZ Travel Portal

نظام إدارة بيانات شركة AZ Travel للسياحة

## ✨ المميزات

- 🌙 وضع ليل/نهار
- 🌐 عربي + إنجليزي
- 🔐 تسجيل دخول بصلاحيات (Admin / Viewer)
- 📊 Dashboard بإحصائيات شاملة
- 📈 تحليلات بـ 6 أنواع Charts
- 🔍 فلاتر قوية + بحث + ترتيب + تصدير CSV
- 👥 إدارة مستخدمين + Audit Log
- 📱 متجاوب مع جميع الشاشات
